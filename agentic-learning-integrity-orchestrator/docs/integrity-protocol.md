# Safety and integrity protocol

## Niveles

- Nivel 1: observacion pedagogica
- Nivel 2: alerta de revision
- Nivel 3: integridad comprometida
- Nivel 4: circuit breaker

## Circuit breaker

Se activa cuando la evidencia no permite una decision justa o cuando convergen senales fuertes de incoherencia, falta de proceso o dependencia improductiva de IA.
