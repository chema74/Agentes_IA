# ROI del agente

Este modulo permite ajustar hipotesis economicas sin recurrir a cifras opacas.

## Variables editables

- horas actuales dedicadas a preparacion de auditoria
- numero de controles por revision
- numero de evidencias por revision
- coste/hora
- tiempo medio de recogida de evidencia
- tiempo medio de revision y seguimiento
- porcentaje de reutilizacion de evidencias
- numero de auditorias mensuales
- coste mensual de implantacion o servicio

## Ejecucion

```powershell
python -m roi.calculator
```

## Principio de uso

El ROI es una aproximacion de eficiencia operativa, no una promesa comercial cerrada. Los datos deben adaptarse al contexto del cliente, al volumen real de controles y a la calidad del gobierno documental existente.
