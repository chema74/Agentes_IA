# Datos demo

Los datos de esta carpeta son anonimos y estan preparados para demostraciones y pruebas locales.

## Contenido

- `controls/controls_demo.csv`: catalogo base de controles
- `evidence/policy_access_review.txt`: procedimiento o politica simulada
- `evidence/network_diagram.png`: referencia visual simulada
- `logs/auth_review_log.txt`: log simple para el flujo de evidencia
- `tickets/security_tickets.csv`: export de tickets para remediaciones y seguimiento
- `users/README.md`: usuarios de acceso demo
