# Change Process Coaching Orchestrator

Orquestador API-first para leer senales de cambio, detectar friccion, planificar intervenciones y escalar a supervision humana cuando corresponde.

## API

- `GET /api/health`
- `POST /api/change-cases/evaluate`
- `POST /api/change-cases/intervene`
- `GET /api/change-cases/{case_id}`
- `GET /api/audit/{reference}`
