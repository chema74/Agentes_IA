Como plantear bien la conversacion

1. Problema concreto

Describe el cuello de botella, no solo la tecnologia que te interesa usar.

2. Material disponible

Di si ya existen PDFs, base de conocimiento, datos tabulares, historicos o procesos documentados.

3. Nivel esperado

Aclara si buscas una demo, un prototipo interno o una pieza con vocacion de producto.

Que devuelve esta web
