# Geopolitical Trade Intelligence Agent

Agente API-first para traducir señales geopolíticas, regulatorias y logísticas en decisiones operativas de comercio internacional.

## API

- `GET /api/health`
- `POST /api/trade-cases/evaluate`
- `POST /api/trade-cases/assess-route`
- `GET /api/trade-cases/{case_id}`
- `GET /api/audit/{reference}`
