# Safety protocol

## Circuit breaker

Se activa cuando aparecen senales compatibles con:

- ideacion suicida
- autolesion
- riesgo inmediato
- abuso severo
- desorganizacion cognitiva grave

## Efectos

- suspender apoyo conversacional ordinario
- priorizar contencion y ayuda humana inmediata
- reducir exploracion abierta
- registrar `HIGH_RISK_ESCALATION`
