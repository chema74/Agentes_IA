# Autonomous Legal Counsel Agent

Agente API-first para triage, scoring, redlining y negociacion contractual B2B dentro de playbooks autorizados.

## API

- `GET /api/health`
- `POST /api/contracts/review`
- `POST /api/contracts/redline`
- `POST /api/contracts/negotiate`
- `GET /api/reviews/{review_id}`
- `GET /api/audit/{reference}`
